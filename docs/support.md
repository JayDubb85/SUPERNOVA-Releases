# Support

Use GitHub Issues for public bug reports and support requests:

[SUPERNOVA Issues](https://github.com/JayDubb85/SUPERNOVA-Releases/issues)

When reporting an issue, include:

- SUPERNOVA version
- Operating system and processor type
- What you were trying to do
- What happened
- Any visible error message

Do not include protected health information, patient identifiers, private PACS credentials, or real DICOM files in public issues.
